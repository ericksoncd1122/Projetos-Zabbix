# Projetos Zabbix

Repositório voltado à publicação de **templates prontos para importação no Zabbix**, desenvolvidos para o monitoramento de **diversos equipamentos, modelos e fabricantes**.

A proposta deste projeto é centralizar templates úteis para ambientes reais de operação, facilitando a implantação do monitoramento com recursos como:

- descoberta automática;
- itens de coleta;
- triggers de alerta;
- gráficos;
- dashboards;
- documentação de apoio.

Este repositório foi criado para servir como base prática para profissionais que atuam com **NOC, provedores de internet, redes corporativas, infraestrutura e monitoramento**, oferecendo materiais organizados e prontos para acelerar a implementação de supervisão no Zabbix.

## Objetivo

Disponibilizar templates funcionais, testados e estruturados para reduzir o tempo de configuração, padronizar a monitoração e contribuir com a comunidade técnica que utiliza o Zabbix no dia a dia.

## O que você encontrará aqui

Neste repositório serão disponibilizados templates para monitoramento de diferentes tipos de equipamentos, como por exemplo:

- OLTs;
- rádios;
- switches;
- roteadores;
- equipamentos de acesso e transporte;
- dispositivos com suporte a SNMP e outras formas de coleta compatíveis com o Zabbix.

## Evolução do projeto

Os projetos e templates deste repositório serão adicionados gradualmente, conforme forem sendo **construídos, validados e testados em ambiente real**.  
A ideia é publicar materiais que realmente tenham utilidade prática no dia a dia operacional, mantendo um acervo cada vez mais completo e confiável para a comunidade.

## Sobre o projeto

Esta é uma **iniciativa privada**, criada com o propósito de **ajudar a comunidade** a monitorar diversos equipamentos no Zabbix de forma mais simples, prática e padronizada.
